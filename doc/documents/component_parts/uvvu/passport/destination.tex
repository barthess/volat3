\point {\nameDevice} предназначен:
  \begin{itemize}
    \item для сопряжения вычислителя бортового ВБ с устройством
      ввода-вывода и управления УВВУ, видеомонитором 1, 
      видеомонитором 2, блоком измерительным спутниковым 
      БИС-2, панелью сигнализации и управления ПСУ;
    \item для подачи напряжения электропитания в вычислитель бортовой ВБ;
    \item для подключения внешних периферийных  устройств во 
      время технического обслуживания (мышь, клавиатура, 
      запоминающие устройства USB, локальная сеть).
  \end{itemize}
